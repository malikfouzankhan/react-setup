1. Clone your github repo
2. Move into the local folder
3. Create a client folder and move into it


4. Run this command: pnpm create vite@latest .
    Choose:
        Framework: React
        Variant: JavaScript

5. Clean these files:
    eslint.config.js
    src/assets/react.svg
    src/App.css
    src/App.jsx
    src/index.css
    src/main.jsx

6. Create these files using the command: touch src/App.jsx src/App.css src/index.css src/main.jsx

7. Run this command: pnpm install && pnpm up --latest
8. Command: pnpm add tailwindcss @tailwindcss/vite

9. Inside src/index.css, write this: 
                                    @import 'tailwindcss'

10. Configure vite for tailwind
    Inside vite.config.js,
                            import {defineConfig} from 'vite';
                            import react from '@vitejs/plugin-react';
                            import tailwindcss from '@tailwindcss/vite';

                            export default defineConfig({
                                plugins: [react(), tailwindcss()],
                            })

11. In src/main.jsx
                    import { createRoot } from 'react-dom/client';
                    import './index.css';
                    import App from './App.jsx';

                    createRoot(document.getElementById('root')).render(<App />);

12. In src/App.jsx
                    import React from 'react';

                    const App = () => {
                        return (
                            <h1 className="text-3xl font-bold text-center text-gray-800 mt-10>
                            Setup is complete!
                            </h1>
                        );
                    };

                    export default App;

13. To run the project: pnpm dev
